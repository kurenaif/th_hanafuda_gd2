参加条件
誰でも参加可能。
主催(f.くれなゐ)と一度も話したことがなくても大丈夫です！

イラストの制約
東方キャラ と 花札の名称に書かれているものは描く
    例:桐に鳳凰×妹紅+慧音 → 慧音+妹紅(鳳凰の代わり)+桐の花
これさえ守っていただければ何やっていただいてもOKです！
ぜひぜひアレンジしてください！
参考: 前回の合同

印刷について
名刺サイズ(91x55mm, 350dpi)
フルカラー(CMYK)
背面にはロゴが入ります
別冊子に皆様の名前等の情報を載せる予定です

作業配信や途中過程，完成品のSNS等への投稿について
じゃんじゃんしてください よろしくお願いします！
ご自身で描かれたイラストに関して，生放送や投稿する際の規定は一切設けません。
完成直後に(頒布前後にかかわらず)原寸大でアップロードしていただいて構いません。
もちろんご自身の作品を，同人誌や同人グッズ，ポートフォリオ等に使用しても構いません．
ただし，他人が描いたイラストを無断で使用することは禁止します．

報酬等
申し訳ありません。 現在金銭的な報酬は考えておりません。
最低限、完成した合同1部または2部、
余裕があればノベルティを作成します。

期限
イラストの提出締め切りは2018年9月1日を予定しています(予定ですがこれより早めることはしません)
頒布は紅楼夢を予定しております。
